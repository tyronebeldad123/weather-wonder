WeatherWonder is a sleek, user-friendly weather app built with **React**, **Vite**, and **Tailwind CSS**.  
It allows users to search for cities worldwide and view current weather conditions, including temperature, wind speed, weather description, and even a weather icon. The app also keeps a **search history** for quick access to previously searched cities.

---

## Features

- Search for any city in the world
- Displays:
  - City and country
  - Temperature in Celsius
  - Wind speed
  - Weather description
  - Weather icon (e.g., â˜€ï¸, ğŸŒ§ï¸)
- **Search History**:
  - Saves previously searched cities
  - Click a history button to quickly view that city again
  - Optionally clear all search history
- Local caching: avoids repeated API calls for the same city for 10 minutes
- Responsive, mobile-friendly layout
- Beautiful design with **Tailwind CSS** and gradient background
